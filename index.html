Raycast Lighting

An example of 2D lighting using only raycasts and knowledge of vertices.

This is a 2D implementation of ray casting in order to create the illusion of sight. You can click in one spot, then another to create a new wall. If you hold Shift while doing the second click, it will begin a new wall in the same position, to make seamless angled walls. If you hit Space on the second-to-last line of a continuous path, it will finish it for you, to make a seamless polygon. The above "Clear Walls" button will erase all walls currently on the board, including the default ones used for demo purposes.
